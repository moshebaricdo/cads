# CADS visual recipes

Store one `<Component>.json` recipe here for every Figma-mapped CADS component.
Start from:

```text
.cursor/skills/cads-parity-qa/visual-recipe-template.json
```

Recipes are evidence captured from the exact Figma component-set node, not
defaults inferred from MUI or the existing implementation. Follow the
`cads-parity-qa` skill for required coverage, deterministic browser captures,
the fix-and-recapture loop, and accepted-difference records.

Do not mark placeholder values as complete. A component without a populated
recipe remains pending parity adoption.
